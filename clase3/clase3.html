Clase 3: Tablas

Para trabajar con tablas usamos la etiqueta table.
Las filas se declaran mediante la etiqueta tr (table row).
Las columas se declaran mediante la etiqueta td (table data).

La estructura para las tablas es que los td van dentro de los tr

1,1 1,2
2,1 2,2

Cabecera y cuerpo de las tablas

Hay 2 etiquetas para definir la cabecera y el cuerpo de una tabla.
thead: para la cabecera.
tbody: para el cuerpo.

Para definir las filas en el thead usamos tr.
Para definir las columnas en el thead usamos th.

Para definir las filas en el tbody usamos tr.
Para definir las columnas en el tbody usamos td.

Combinacion de celdas

Para combinar una o varias columnas usamos la propiedad o atributo colspan.
Este atributo recibe un número que indica la cantidad de columnas a combinar

Para combinar una o varias filas usamos la propiedad rowspan.
Este atributo recibe un número que indica la cantidad de filas a combinar
