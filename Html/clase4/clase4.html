Formularios: form

Nos sirven para recolectar información del usuario.

Para definir un formulario usamos la etiqueta form

Elementos

Las etiquetas para crear elementos de formularios son input, select, textarea, button

Atributos

Method

Especifica el método HTTP (GET, POST)

Name

Es un atributo importante para el envio de formularios. Es el nombre que toma el input para capturar el dato y enviarlo
