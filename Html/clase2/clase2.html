Listas

Para declarar una lista usamos la etiqueta ol o ul, y para los elementos de las listas usamos li

Listas ordernadas (ol)

Son listas que tienen un orden

  1. Primer elemento
  2. Segundo elemento
  3. Tercer Elemento

Listas desordernadas (ul)

Son listas que NO tienen un orden

  • Elemento Y
  • Elemento x
  • Elemento Z

Lista anidadas

Listas dentro de listas

  • Numeros:
    1. Uno
    2. Dos
    3. Tres
    4. Cuatro
    5. Cinco
  • Letras:
    1. A
    2. B
    3. C
    4. D
    5. E
  • Vocales:
    • A
    • E
    • I
    • O
    • U
