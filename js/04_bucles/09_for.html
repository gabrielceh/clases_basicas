Bucle for

Esta es la estructura repetitiva mas comun para realizar bucles.
Consta de 3 partes:
Declaracion 1: es ejecutada antes de que bucle comience
Declaracion 2: Es la condicion para ejecutar el bucle
Declaracion 3: Se ejecuta cada vez y despues que el codigo ha sido ejecutado. 

        for(let i = 0; i <= 10; i++){
          //codigo
        }
