Variables y constantes

Variables: var vs let

Anteriormente se declaraban variables usando la palabra var. Pero habia un problema, puesto que var tenia un alcance global o en la funcion donde estaba definida, pero no era por bloque de codigo como en otros lenguajes de programacion.
A partir de EM6, ya se pueden declarar variables de ambito bloque, para ello usamos la palabra let.

Constantes

Fueron instroducidas en EMC 6.
Las constantes se usan cuando sepamos que una variable no va a cambiar su valor durante el flujo de la app.
No se pueden tener constantes vacias, es decir, se le debe asignar su valor al momento de ser declarada.
Despues de ser asigando su valor, este no podrá ser modificado, si es un dato primitivo(number, string, boolean ).
