Operadores aritmeticos - comparacion - logicos

Operadores aritmeticos

Son los que realizan funciones aritmeticas en numeros

Operador Descripcion Ejemplo
+ Adicion 3 + 5 = 8
- Substracion 9 - 5 = 4
* Multiplicacion 10 * 5 = 50
/ Division 21 / 3 = 7
% Modulo. Retorna el resto de la division 29 / 3 = 2
++ Incremento. Aumenta el valor numerico en 1. let num = 1 ; num++ ; num ahora vale 2
-- Decremento. Disminuye el valor numerico en 1 let num = 2; num--; num ahora vale 1

Operadores de comparacion

Son utilizados en declaraciones logicas para determinar igualdad o diferencia entre variables o valores. Estos retornan true o false.

Operador Descripcion Ejemplo
== Igual que 5 == 10 false
=== Identico 5 === '5' false
!= No es igual 5 != 'a' true
!== No es identico 10 !== '10' true
> Mayor que 4 > 2 true
< Menor qie 4 < 2 false
>= Mayor igual que 5 >= 5 true
<= Menor igual que 5 <= 4 false

Operadores Logicos

Evaluan una expresion y retornan true o false

Operador Descripcion
&& (Y): retorna true si ambas expresiones son verdaderas
|| (O): retorna true si una de las expresiones es verdader
! (NO): retorna true si la expresion es falsa, y false si la expresion es verdadera
