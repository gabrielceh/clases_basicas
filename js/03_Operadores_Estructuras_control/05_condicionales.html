Condicionales (if)

A menudo cuando se escribe codigo, se desea ejecutar diferentes acciones basado en diferentes condiciones. Para ello utilizamos las condiciones en el codigo.
Se utiliza la declaracion if para especificar un bloque de codigo que se ejecutará su la condicion es verdadera.

Declaracion if-else

La declaracion else ejecutara un bloque de codigo si la declaracion del if es falsa.

declaracion else-if

Especifica una nueva condicion si la primera o la anterior es falsa
