Funciones

Una funcion es un bloque de codigo autocontenido, que se definirá una vez y se reutilizara las veces que sea necesarias.
La herramienta más potente con que se cuenta para facilitar, reducir y dividir el trabajo en programación, es escribir aquellos grupos de sentencias una sola y única vez bajo la forma de una FUNCION.
Las funciones en js tienen la sgte estructura: 

        function nombreDeLaFuncion () {
          // codigo a ejecutar 
        }

Al momento de nombrar una funcion, debemos tener en cuenta cual sera el objetivo de esta para que sepamos con mas claridad que es lo que hara ese bloque de codigo.
Para ejecutar una funcion es necesario llamarla. Para ello, comienza con el nombre de la funcion, luego con los argumentos si es que tiene.
Las funciones pueden o no recibir parametros y pueden o no retornar un valor.

Funciones con parametros

Reciben en sus parentisis una o varios parametros que podemos usar dentro de la funcion.
Al igual que las variables, a los parametros se les debe dar nombres, que se separan por comas dentro de los parentesis. 

        function nombreDeLaFuncion (parametro1, parametro2,...) {
          // codigo a ejecutar 
        }

Cuando llamamos a la funcion, le proveemos el valor del parametro, llamado argumento, entre parentesis. Cuando la funcion recibe varios parametros, los argumentos deben pasarse en el orden en el que los definiste.

Parametros por defecto

Se le pueden asignar parametros por defecto a una funcion, igualando el parametro a un valor. 

      function nombreDeLaFuncion (parametro1, parametro2 = valor) {
        // codigo a ejecutar 
      }

Funciones que retornan un valor

La sentencia return finaliza la ejecución de la función y especifica un valor para ser devuelto a quien llama a la función.
Esta declaracion (return) es util cuano hacemos calculos que requieran un resultado. Si no se retorna ningun valor desde la funcion, esta devolverá undefined. 

        function nombreDeLaFuncion (parametro1, parametro2,...) {
          // codigo a ejecutar 
          return valorARetornar
        }

Funciones declaradas y funciones expresadas

Las funciones declaradas son las que hemos venido trabajando hasta el momento. Estas funciones tienen la particularidad de poder llamarse incluso antes de haber sido declaradas. Esto es a causa del hoisting como sucede con var.

Las funciones expresadas son funciones que se le asigna como valor a una variable o constante, por lo general a constantes, ya que su valor como funcion no cambiará. NO pueden invocarse antes de ser declaradas

      const nombreFuncion = function(){
        //...
      }
