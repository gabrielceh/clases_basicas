String - Number - boolean - undefined - null - isNaN

Para conocer cual es el tipo de dato, usamos typeof()

string

Son cadenas de texto. Pueden ser creadad usanado comillas simples o dobles.
Una forma de declararla es con su constructor 

let algo = new String("texto")

Metodos mas comunes

string.length
Longitud de la cadena
string.toLowerCase()
Pasa la cadena a minusculas
string.toUpperCase()
Pasa la cadena a mayusculas
string.includes("...")
devuelve true o false si encuentra o no una coincidencia en la cadena
string.trim()
Quita los espacios en blancos al inicio y al final de la cadena
string.split()
Divide la cadena segun la expresion regular que le pasemos y la convierte en un arreglo
string.charAt(indice)
Retorna el caracter del indice que le indiquemos
string.substring(pos.inicio, pos.fin)
Retorna un string extraido de otro. La posicion de inicio es el indice del caracter donde queremos que inicio. la posicion final es el indice del caracter donde queremos que termine pero sin incluirlo
string.indexOf("...")
Devuelve la posición de la subcadena dentro de la cadena, o -1 en caso de no estar.
string.replace(subcad1, subcad2)
Reemplaza la subcadena 1 por la subcadena 2

Template string

La interpolacion es introducir dinamicamente una variable en una cadena de texto. En js usamos template string para esto.
Para ello usamos back stick ` ` y para colocar las variables dentro usamos ${nombreVariable} dentro de las back sticks

numbers

Solo tenemos numeros en js, sin importar si son enteros, decimales.
Al igual que los strings, lo numeros tambien tienen metodos

number.toFixed()
redondea un numero segun la cantidad de numeros que le pasemos
parseInt()
convertirá la cadena en un numero entero
parseFloat()
Convertira la cadena en un numero flotante
parseFloat()
Convertira la cadena en un numero flotante
number.valueOf()
Retorna el valor primitivo inserto en un objeto Number

booleans

Representan los valores de true o false

Hay valores que no son true o false pero que tienden a ello

  • 1 = true
  • 2 = true
  • 0 = false
  • "hola" = true
  • "" = false

undefined - null - isNaN

undefined
representa una variable sin valor y que no ha sido inicializada
null
Representa una variable sin valor pero es intencionalmete inicializada con null
isNaN
not a number, nos sirve para validar si es un numero o no.
