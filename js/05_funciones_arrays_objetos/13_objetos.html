Objetos

Vemos a las variables como lugares donde podemos contener datos o valores. Los objetos son tambien variables, pero estos pueden contener muchas mas variables e incluso funciones, arrays y otros objetos.
Para simplificar, los objetos con como una lista de valores, que son escritos como pares de llave:valor. A estos valores los llamamos propidades. 

        //creacion de objetos
        const persona = {
          nombre: "Gabriel",
          edad: 31,
          nacionalidad: "Colombiano",
          "ciudad-nacimiento":'Barranquilla',
          pasatiempos: ['Videojuegos', 'Anime', 'Programar'],
          soltero: true,
          twitter: "@gabrielcehu",
          instagram: "@gabo_cervantes",
          saludar: function(tiempo='buenos dias'){
            console.log(`Hola, ${tiempo}, soy ${this.nombre}`)
          }
        }

Para acceder a las propiedades, podemos hacerlo de dos maneras: 

        //notacion del punto:
        objeto.propiedad
        ej: personas.nombre

        //notacion de corchetes
        objeto['propiedad']
        ej: persona['nombre']

Dentro de los objetos, a las variables se les llama atributos y a las funciones se les llama metodos.
Si queremos acceder al atributo dentro de una funcion, accedemos mediante la palabra this. ej: this.nombre

Incluso es posible hacer que el valor de un miembro del objeto sea otro objeto. 

        //reemplacemos las redes sociales
        const persona = {
          nombre: "Gabriel",
          edad: 31,
          nacionalidad: "Colombiano",
          "ciudad-nacimiento":'Barranquilla',
          pasatiempos: ['Videojuegos', 'Anime', 'Programar'],
          soltero: true,
          redes:{
            twitter: "@gabrielcehu",
            instagram: "@gabo_cervantes",
          },
          saludar: function(tiempo='buenos dias'){
            console.log(`Hola, ${tiempo}, soy ${this.nombre}`)
          }
        }

        //para llamarlas podemos hacer lo sgte:
        persona.redes.twitter
        o
        persona['redes']['twitter']

Tambien podemos modificar propiedades o crearlas. 

        //modificarlas
        persona.soltero = false
        o
        persona['soltero'] = false

        //crearlas 
        personas.apellido = 'Cervantes'
        o
        persona['apellido'] = 'Cervantes'

Metodos mas comunues de los objetos

Object.key()

Recibe un objeto y muestra todas las llaves(keys) del objeto

Object.values()

Recibe un objeto y muestra todas los valores(values) del objeto

Object.entries()

Recibe un objeto y devuelve todad los pares llaves:valor en forma de array

hasOwnProperty()

Recibe una propiedadd del objeto y verifica si este la posee o no.
