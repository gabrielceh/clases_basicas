Animation

Permite a un elemento cambiar de un estilo a otro. Puedes cambiar de propiedades Css como quieras las veces que quieras.

Para especifiar una animacion, usamos la sgte sintaxis:

      elemento{
        animation-name: nombreAnimacion;
        animation-duration: tiempo en segundo;
      }

      @keyframes nombreAnimacion {
        from{
          propiedad: valor;
          ...
        }
        to{
          propiedad: valor;
          ...
        }
      }

animation-iteration-count - animation-direction

Si queremos que la animacion se repita, usamos animation-iteration-count.

animation-direction especifica como debe repetirse la animacion. Si le indicamos normal, cada que finaliza la animacion, inicia desde el principio. Si le indicamos "alternate", cuando finzaliza comienza desde el final hasta el inicio

animation-timing-function - animation-delay

animation-timing-function, es similar a las transiciones

  • ease: la animacion comienza lento y luego acelera
  • ease-in: la animacion comienza lento y luego acelera y se detiene abruptamente
  • ease-out: la animacion comienza rapido y luego desacelera hasta detenerse
  • ease-in-out: simialar a ease pero mas suave
  • linear: constante

animation-delay retrasa el inicion de la animacion

animation-play-state

nos permite cambiat el estado de la animacion, sus valores posibles son "paused" y "running", esta ultima es por defecto
