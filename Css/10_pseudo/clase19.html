Pseudo-elementos

Son usados para seleccionar partes especificas de un elemento

Hay 5 pseudo elementos en css

  • ::first-line: primera linea de un texto
  • ::first-letter: primera letra de un texto
  • ::selection: selecciona una porcion de un elemento seleccionado por el usuario
  • ::before: inserta contenido antes del elemento
  • ::after: inserta contenido despues del elemento
