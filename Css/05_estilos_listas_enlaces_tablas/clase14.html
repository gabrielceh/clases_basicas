Estilos enlaces

Se le puede cambiar el estilo de la misma forma que se hace con los texto. Pero adicionalmente podemos usar pseduo-clase

  • :link - define el estilo cuando enlace no ha sido visitado
  • :active - define el estilo cuando enlace se hace click
  • :visited - define el estilo cuando enlace ha sido visitado
  • :hover - define el estilo cuando pasas el cursor sobre el enlace
