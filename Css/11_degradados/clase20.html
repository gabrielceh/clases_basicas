Degradados

Degradados lineales

Debemos definir al menos 2 paradas de colores. Las paradas son los colores con los que quieres crear el degradado.

Para trabajar con degradados lineales usamos linear-gradient(color1, color2)

Las paradas de cada color se pueden especificar. Se pueden usar %, px, rem, etc. linear-gradient(color1 20%, color2 80%).

La direccion del degradado puede ser modificado. Puedes usar left, bottom, right, top o sus combinaciones, ejemplo left top. linear-gradient(left, color1, color2)

Podemos indicar la direccion tambien especificando un angulo. Un angulo de 0° va de izquierda a derecha. Un angulo de 90° va de abajo hacia arriba. linear-gradient(100deg, color1, color2)

Degradados radiales

Debemos definir al menos 2 paradas de colores. Las paradas son los colores con los que quieres crear el degradado.

Para trabajar con degradados lineales usamos radial-gradient(position, forma o tomaño, colore1, color2)

  • posicion: Define la posicion del degradado mediante % o las palabras para posicionar(top, left, bottom, right)
  • forma o tamaño: define la forma y el tamaño del degradado. Para darle forma tenemos 2 argumentos: ellipse(por defecto), circle
