display, visibility y position

Display

Cada elemento en una pag web es una caja. La propiedad display determina como se comporta esa caja.

display: block

Un elemento en bloque ocupa el ancho maximo posible, con saltos de linea antes y despues.

display:none

Oculta el elemento, haciendo que este no ocupe espacio. La pag se ejecutará como si el elemento no existiera

visibility

Especifica si un elemento es visile o está oculto. Se diferencia de display:none, en que este, el elemento no ocupara espacio, en cambio, con visibility si ocupara espacio en la pag.

Sus valores mas comunes son: visible y hidden

position

Te permite posicionar un elemento.

Para posicionar los elemenos usamos las propiedades: top(arriba), bottom(abajo), left(izq.), right(derecha)

position:static

Por defecto, los elementos son posicionados de forma estatica. Cuando se posicionan asi, siempre se pocisiona de acuerdo al flujo del documento
